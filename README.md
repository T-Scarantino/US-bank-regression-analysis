# Replication analysis on US bank recovery post recessions
## project overview
This project is a replication analysis evaluating whether US banks have recovered from past recessions using more recent data. It uses updated data and multiple regression to understand long-term financial performance trends.
## objective
 the goal of this project is to compare previous results on US bank performance before and after recessions with more recent data to see if the results are consistent
## data
the full raw data set used in this project was too large for GitHub
- full raw dataset available via external link: [https://drive.google.com/file/d/1iEk3OdZm7r-gAaVuvhbpda1KTM1UKORN/view?usp=sharing]
## methods
- data cleaning and aggregation was performed prior to the analysis
- the large dataset was summerized using group averages
- multiple linear regression conducted using R
